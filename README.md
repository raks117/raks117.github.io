Recycled HDPE & PP Granules Manufacturer, India.

## 🚀 How to Go Live FREE on GitHub Pages (Step-by-Step)

### Step 1 — Create a GitHub Account
Go to https://github.com and sign up (free).
Suggested username: **skplasticworks**

### Step 2 — Create a New Repository
1. Click **"New repository"**
2. Repository name: `skplasticworks.github.io` ← exact name matters
3. Set it to **Public**
4. Click **"Create repository"**

### Step 3 — Upload the Website Files
1. Click **"uploading an existing file"**
2. Drag and drop ALL files from this folder:
   - `index.html`
   - `about.html`
   - `css/style.css`
   - `js/main.js`
   - `sitemap.xml`
   - `robots.txt`
3. Click **"Commit changes"**

### Step 4 — Enable GitHub Pages
1. Go to **Settings → Pages**
2. Under "Source", select **main branch / root**
3. Click **Save**

### Step 5 — Your site is LIVE! 🎉
Visit: **https://skplasticworks.github.io**
(may take 1–2 minutes to appear)
